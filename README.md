﻿# SSSCharacterCreator
An [Exiled](https://github.com/ExMod-Team/EXILED/tree/master/EXILED) plugin for SCP:SL that adds to SSS a character
creator. Best used for RP servers.

![Some version](https://img.shields.io/github/v/release/michcio15/SSSCharacterCreator?style=for-the-badge) ![GitHub License](https://img.shields.io/github/license/michcio15/SSSCharacterCreator?style=for-the-badge)
## Dependencies

[ASS](https://github.com/Someone-193/ASS/) by *@someone* (i used the exiled one for my testing)

## Usage

For player to get the character creator he needs to have the permission stated in config

```yaml
character_creator_permission: 'cc.create'
```
Player can be spawned only if he is **not** alive and the round is started.
